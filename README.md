# WechatSogouJS
基于搜狗微信搜索的微信公众号JS爬虫，目前此项目仅爬取 URL（临时），请在URL有效期内，及时使用其他方式爬取内容。推荐使用
[Chyroc/WechatSogou](https://github.com/Chyroc/WechatSogou) 的Python爬虫做进一步的获取。
